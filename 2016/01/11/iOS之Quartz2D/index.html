iOS之Quartz2D

发表于 2016-01-11   |   分类于 iOS   |  

这部分主要是摘录于文顶顶大大的博客。文顶顶
记录自己的学习。

what is Quartz2D

他是一个二维绘图引擎。

他能干什么

  1. 绘制图形 : 线条\三角形\矩形\圆\弧等
  2. 绘制文字
  3. 绘制\生成图片(图像)
  4. 读取\生成PDF
  5. 截图\裁剪图片
  6. 自定义UI控件

    iOS中⼤部分控件的内容都是通过Quartz2D画出来的

    自定义view

  7. 新建⼀个类,继承自UIView
  8. 实现-(void)drawRect:(CGRect)rect⽅法.然后在这个⽅方法中 :
    • 取得跟当前view相关联的图形上下文
    • 绘制相应的图形内容
    • 利用图形上下文将绘制的所有内容渲染显示到view上面

      drawRect

  9. 为什么要实现drawRect:⽅法才能绘图到view上?
    • 因为在drawRect:⽅法中才能取得跟view相关联的图形上下文

  10. drawRect:⽅法在什么时候被调用?

    • 当view第一次显示到屏幕上时(被加到UIWindow上显示出来)

    • 调用view的setNeedsDisplay或者setNeedsDisplayInRect:时

      例子

      -(void)drawRect:(CGRect)rect {
      // 1、取得图形上下文
      CGContextRef ctx = UIGraphicsGetCurrentContext();
      // 2、绘图
      // 设置绘图起点
      CGContextMoveToPoint(ctx, 20, 20);
      // 绘制线段到指定坐标
      CGContextAddLineToPoint(ctx, 200, 100);

      // 3、设置绘图状态
      // 设置线段颜色
      CGContextSetRGBStrokeColor(ctx, 0, 1, 1, 1);
      // 设置线条宽度
      CGContextSetLineWidth(ctx, 10);
      // 设置线段起点和终点的样式为圆角
      CGContextSetLineCap(ctx, kCGLineCapRound);
      // 绘制
      CGContextStrokePath(ctx);

      // 绘制三角形
      CGContextMoveToPoint(ctx, 20, 100);

      //设置第二个点

      CGContextAddLineToPoint(ctx, 40, 300);

      //设置第三个点

      CGContextAddLineToPoint(ctx, 200, 200);

      //设置终点
//     CGContextAddLineToPoint(ctx, 20, 100);

      //关闭起点和终点

      CGContextClosePath(ctx);

//    空心圆
CGContextMoveToPoint(ctx, 150, 400);
CGContextAddArc(ctx, 100, 400, 50, 0, 2 * M_PI, 0);

// 3.渲染图形到layer上
CGContextStrokePath(ctx);
//    实心圆
CGContextAddEllipseInRect(ctx, CGRectMake(250, 100, 50, 50));

[[UIColor greenColor] set];

// 3.渲染
//    CGContextStrokePath(ctx);
CGContextFillPath(ctx);
