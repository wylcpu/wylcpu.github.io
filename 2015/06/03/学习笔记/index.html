  1. 实现元素水平垂直居中二

代码

<style>
.box{
    width:300px;
    height: 300px;
    border: 1px solid red;
    position: relative;
}
.element{
  position: absolute;
  width: 100px;
  height: 100px;
  left: 50%;
  top: 50%;
  margin-left: -50px;
  margin-top: -50px;
  background-color: yellow;
}
</style> 
<body> 
<div class="box">
    <div class="element">dfd</div>
</div>
</body>
  1. 实现元素水平垂直居中方法三

代码

  left: 50%;
  background:yellow;
  transform: translate(-50%,-50%);

    width:300px;
    height: 300px;
    border: 1px solid red;
    position: relative;
}
</style>
<body>

    <div class="element">dfd</div>
</div>
