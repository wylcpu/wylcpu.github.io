html之input

htmlhtml5
  • html
  • html5
  • input

今天是端午节，苦逼在敲代码

#input

###type属性
type规定了input元素的类型

<input type="text"/>
这个是最常见的输入文本类型，这个只能输入单行文本，通过获取元素的value来获取其中的值

<input type="button"/>
这个定义了一个按钮，但是没有任何行为，就是一个单独的按钮，这和submit不同。你可以为它指定点击事件
<input type="checkbox/>"
定义一个复选框，你可以多选，选择一个或者是多个，用name属性来指定这是不是同一类复选框，这个属性一定要有的。value属性，来指定input元素的值
<input type="file"/>
用于文件上传
<input type="hidden"/>
定义一个隐藏的字段，隐藏一个input标签，可以通过js获取到
<input type="image"/>
定义了一个图片式的提交按钮，这个一定要有’src’属性来指定一个图片的URL，最好还有一个alt，当图片不能显示时，显示的内容
<input type="password"/>
这个是用于输入密码的，输入的密码会被星号或者是点号隐藏
<input type="radio"/>
定义一个单选框，你可以定义一定数目的单选框，只能选择一个，用name来指定同一类，value指定每个单选框的值
<input type="reset"/>
定义一个重置按钮，它会清空所有表单中的数据，这个是要有form来指定是一个表单下,她会清除form中的input中的数据
<input type="submit"/>
定义一个提交按钮，默认会发送form表单中的数据

###value属性
不同的type属性，value属性表示的也不同

  • type=”button”, “reset”, “submit” - 定义按钮上的显示的文本
  • type=”text”, “password”, “hidden” - 定义输入字段的初始值
  • type=”checkbox”, “radio”, “image” - 定义与输入相关联的值，也就是点击那个按钮的值
    <input type="checkbox"> 和 <input type="radio"> 中必须设置 value 属性
    <input type="file">不能有value

###name属性
name 属性用于对提交到服务器后的表单数据进行标识，或者在客户端通过 JavaScript 引用表单数据，
只有设置了 name 属性的表单元素才能在提交表单时传递它们的值，这个要必须有，除非你用js单独获取它们的值用Ajax发送

###size属性
对于 <input type="text"> 和 <input type="password">，size 属性定义的是可见的字符数。而对于其他类型，size 属性定义的是以像素为单位的输入字段宽度

###maxleng属性
用于text和password类型的，规定了可以输入的最多字符数，也就用于这两种情况

###checked属性
用于radio和checkbox类型，有它代表，预先指定选中的状态

###disabled
被禁用的 input 元素既不可用，也不可点击，不能用于type是hidden类型,通过js来取消

#form
可以包含input,textarea,fieldset,legend,label;

###enctype
application/x-www-form-urlencoded 在发送前编码所有字符（默认）
multipart/form-data 不对字符编码。
在使用包含文件上传控件的表单时，必须使用该值。
text/plain 空格转换为 “+” 加号，但不对特殊字符编码。

###action
指定表单提交到服务器的URL

###method
指定表单提交的方式GET，POST

###accept-charset
accept-charset 属性规定服务器用哪种字符集处理表单数据

###target

  • _blank 在新窗口中打开。
  • _self 默认。在相同的框架中打开。
  • _parent 在父框架集中打开。
  • _top 在整个窗口中打开。
  • framename 在指定的框架中打开。

#fieldset
用于将表单用的元素进行分组，当一组表单元素放到

标签内时，浏览器会以特殊方式来显示它们，它们可能有特殊的边界、3D效果，或者甚至可创建一个子表单来处理这些元素
用legend元素来指定标题

#legend
用于给fieldset元素指定标题

#label
可以与input元素绑定在一起，当你点击相应的label就像点击对应的input是一样的，他们的绑定是通过，label的for属性，

#textarea
标签定义多行的文本输入控件,宽高可以用cols，rows设定

#select
下拉式菜单栏，用option元素指定下拉菜单的数目

###selected
selected="selected"
定义预先为选中的状态

###multiple
multiple="multipe"可以进行多选

###size
规定可见option元素的可见数
