java学习笔记（2）

发表于 2015-07-12   |   分类于 java   |  

今天是学习Java的第二天，虽然明天就要考试了，但是我还是想学！今天就学学Java的变量和常量！

java关键字


java是区分大小写的

什么是标识符

标识符就是用于给 Java 程序中变量、类、方法等命名的符号
在命名的时候需要遵守的规则

  1. 标识符可以由字母、数字、下划线（_）、美元符（$）组成，但不能包含 @、%、空格等其它特殊字符，不能以数字开头。譬如：123name 就是不合法滴
  2. 标识符不能是 Java 关键字和保留字（ Java 预留的关键字，以后的升级版本中有可能作为关键字），但可以包含关键字和保留字。如：不可以使用 void 作为标识符，但是 Myvoid 可以
  3. 标识符是严格区分大小写的。 所以涅，一定要分清楚 imooc 和 IMooc 是两个不同的标识符哦！
  4. 标识符的命名最好能反映出其作用，做到见名知意
    ps.其实其它语言的命名规则也是这样的。

什么是变量

描述一个变量通过三个部分：变量类型、变量名以及变量值

如何命名Java变量

下面是比较好的习惯，当然你也可以不这样

  1. 变量名由多单词组成时，第一个单词的首字母小写，其后单词的首字母大写，俗称骆驼式命名法
  2. 变量命名时，尽量简短且能清楚的表达变量的作用，做到见名知意。
  3. Java 变量名的长度没有限制，但 Java 语言是区分大小写的
使用规则
  1. Java 中的变量需要先声明后使用
  2. 变量使用时，可以声明变量的同时进行初始化

java中的数据类型

Java 语言是一种强类型语言。通俗点说就是，在 Java 中存储的数据都是有类型的，而且必须在编译时就确定其类型

基本数据类型变量存的是数据本身，而引用类型变量存的是保存数据的空间地址

Java中char和String 的区别

  1. char表示字符，定义时用单引号，只能存储一个字符，如char c=’x’;
    而String表示字符串，定义时用双引号，可以存储一个或多个字符，如String name=”tom”;
  2. char是基本数据类型，而String 是一个类，具有面向对象的特征，可以调用方法，如name.length()获取字符串的长度。
    Java中float与double的区别
  3. float是单精度浮点数，内存分配4个字节，占32位，有效小数位6-7位
    double是双精度浮点数，内存分配8个字节，占64位，有效小数位15位
  4. java中默认声明的小数是double类型的，如double d=4.0
    如果声明： float x = 4.0则会报错，需要如下写法：float x = 4.0f或者float x = (float)4.0
    其中4.0f后面的f只是为了区别double，并不代表任何数字上的意义
  5. double 和 float 的区别是double精度高，但double消耗内存是float的两倍，且double的运算速度较float稍慢

Java中的自动类型转换

int a = 82;
double b = a;
System.out.println(b)//82.0

满足自动转换的条件

  1. 目标类型能与源类型兼容，如double 型兼容int 型，但是 char 型不能兼容 int 型
  2. 目标类型大于源类型，如 double 类型长度为 8 字节， int 类型为 4 字节，因此 double 类型的变量里直接可以存放 int 类型的数据，但反过来就不可以了

Java中的强制类型转换

格式：( 数据类型 ) 数值
强制类型转换可能会造成数据的丢失，在应用的时候要慎重！

double avg = 75.8;
int avg2 = (int)avg;
System.out.println(avg);//75.8
System.out.println(avg2);//75

Java常量的应用

常量，我们可以理解为是一种特殊的变量，它的值被设定后，在程序运行过程中不允许改变
格式：final 变量名 = 值;
final String s = "hello";
这就定义了一个常量
注意：常量名一般使用大写字符（习惯问题）

Java注释

Java中注释有三种类型：单行注释、多行注释、文档注释
