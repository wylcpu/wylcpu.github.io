java环境搭建

安装JDK

去官网下载

配置环境变量

  1. JAVA_HOME 配置JDK安装路径
    就是C:\Program Files\Java\jdk1.8.0_45
  2. PATH 配置JDK命令文件的位置
    就是C:\Program Files\Java\jdk1.8.0_45\bin
  3. CLASSPATH 配置类库文件的位置
    就是C:\Program Files\Java\jdk1.8.0_45\lib
    这个需要注意的是变量值是.;C:\Program Files\Java\jdk1.8.0_45\lib,注意前面有个点号和分号，点号代表当前目录，分号是分割符
    查看配置是否正确
    在cmd中，分别输入java和javac如果出现很多关于他们的参数的信息那么你就配置对了

使用记事本编写helloworld

代码

public class HelloWorld{
public static void main(String[] args) {
    System.out.println("hello world");
}
}

为了方便首先cmd到文件的所在目录，在cmd中依次运行javac HelloWorld.java如果你的Java代码编写错误就会报错，如果什么都没有提示那就是没有错，编译正确，会生成一个扩展名为.class的文件，在运行java HelloWorld,这个地方要注意不要带上扩展名
