精通css读书笔记

css
  • css读书笔记

##第一章

  1. 在给HTML标签的id和类进行命名时，要保持名称与表现方式无关。根据‘他们是什么’来命名，而不是根据它们的外观来命名。例如：我想让几个个HTML元素显示红色来表示警告，你不能给这些标签添加一个.red的类，而是应该是一个.warning的类
  2. DTD（文档类型定义）是一组机器可读的规则，它们定义了xml或HTML的特定版本中允许有什么，不允许有什么。在解析网页时，浏览器将使用这些规则检查页面的有效性并且采取相应的措施。浏览器是通过分析页面的DOCTYPE声明来了解使用了哪个DTD，由此知道使用了HTML的哪个版本
  3. 浏览器模式：为了确保向后兼容，浏览器厂商创建了两种呈现模式：标准模式和混杂模式；在标准模式中，浏览器根据标准规范呈现页面；在混杂模式中，页面以一种比较宽松的向后兼容的方式显示。ps，对这两种模式之间的差异，最显著的例子就是IE的专有的盒模型：border-box；（嘿嘿现在css3又用这种方便的盒模型了）

##第二章

###层叠与特殊性：

- 类型选择器，伪元素选择器：1
- 类选择器，伪类和属性选择器：10
- id选择器：100
- 行内样式：1000

通俗说，就是用style属性编写的规则总是比其他任何类型规则特殊。具有ID选择器的规则比没有ID选择器的规则特殊，具有类选择器的规则比只有类型选择器的规则特殊。如果两个规则的特殊性相同，则后定义的规则优先。

##第三章

###外边距叠加
当两个或更多 垂直 外边距相遇时，他们将形成一个外边距。这个外边距的高度等于发生叠加的外边距的高度中较大者。
存在即合理，在由几个段落组成的典型文本页面中。如图，如果不发生叠加会很难看，叠加就保持了段落之间距离的一致性。
上图：
只有普通文档流中块框的垂直外边距才会发生外边距叠加。行内框、浮动框、绝对定位框之间的外边距不会叠加

###定位概述

  • css中有三种基本定位机制：普通流、浮动、绝对定位
  • 行内框可以使用水平的内边距、边框，外边距调整水平间距，但是，垂直内边距、边框、外边距不影响行内框的高度（这个地方要注意了，是不影响行高，而不是没有影响），显示的设置宽高，都是没有影响的。举例子：

        border:2px solid red;

<body>

这个地方还有一个概念：行框，和行内框有什么区别？
由一行形成的水平框称为行框，行框的高度 总是 足以容纳它包含的所有行内框。也就是说行高是由其中的最大的行内框的高度决定的。
相对定位和绝对定位
相对定位是“相对于”元素在文档流中德初始位置，而绝对定位是“相对于”距离它最近的已定位的祖先元素。
浮动
浮动的框左右移动，直到它的外边缘碰到包含框或另一个浮动框的边缘

##第五章

###对链接应用样式

  • 伪类选择器，:link伪类选择器用来寻找没有被访问过的链接，:visited伪类选择器用来寻找被访问过的链接，:hover是用来寻找鼠标悬停处的元素，:active用来寻找被激活的元素.
  • 为了尽可能提高页面可访问性，在定义鼠标悬停状态时，最好再链接上添加:focus伪类，这样在通过键盘（如Tab）键移动到链接上时，这让链接显得样式与鼠标悬停时相同。如：a:hover,a:focus{color:red}
  • 除低版本的IE不支持除链接之外的在其他元素上使用伪类选择器，现代的浏览器都支持。
  • 上面的伪类选择器中，如果在同一个元素上出现，他们的顺序是很重要的。

例如：

<style>
    a:link,a:visited{text-decoration: none;}
    a:hover,a:focus,a:active{text-decoration: underline;}
</style>

这样在鼠标悬停时会显示下划线，但是如果你这样写

<style>
    a:hover,a:focus,a:active{text-decoration: underline;}
    a:link,a:visited{text-decoration: none;}
</style>

当鼠标悬停在元素上时是没有没有下划线的。
这是因为这两个规则具有相同的特殊性，后面的样式覆盖了前面的样式。
为了不发生上面的情况，使用他们的顺序是
a:link,a:visited,a:hover,a:focus,a:active

  • 链接使用#号可以跳转到页面的指定位置

###css精灵
css精灵：包含许多不同的图标、按钮或图形的 单个 图像。

  • 使用这种技术可以减少Web浏览器发出的服务器请求，这会显著加快下载速度。另外把图形集中在一个地方可以提高可维护性。

###纯css工具提示
工具提示：当鼠标悬停在具有title属性的元素上时一些浏览器会弹出小文本框。可以结合js和css创建独特的自定义的工具提示。
但是也可以使用纯css方式实现。

<style>
    a.tooltip{
        position: relative;
    }
    a.tooltip span{
        display: none;
    }
    a.tooltip:hover span{
        display: block;
        position: absolute;
        top:1em;
        left:2em;
        padding:0.2em 0.6em;
        border: 1px solid #963;
        background-color:#ff6;
        color:#000;
    }
</style>
<body>
    <a href="www.eviloo7.com" class="tooltip">who am i<span>i am eviloo7</span></a>
</body>

##第六章：对列表应用样式和创建导航条

  • 为了添加定制的项目符号，可以使用list-style-image属性。但是这种方法对项目符号图像的位置的控制能力不强。更常用的的时关闭项目的符号，并且将定制的项目的符号作为背景添加在列元素上，然后使用背景图像的定位属性精确地控制自定义项目符号的对准方式。
