发表于 2015-06-18   |   分类于 DOM   |  

元素鼠标事件中

offsetX,offsetY

获取的是点击位置距离元素左边界和上边界的距离大小；

clientX,clientY

坐标值是鼠标事件发生时鼠标光标在可视区（视口）的位置（我们看得见页面的地方）

layerX,layerY

这个是很麻烦的，它是获取发生鼠标事件时光标距离最近position设置为absolute或者relative元素的距离（本身，或者是祖先元素）,positioin:fixed时很诡异，父元素设置了position,对其是由影响的

xss初认识

发表于 2015-06-08   |   分类于 xss   |  

作为web人员还是了解一些web安全是有必要的

xss入门

什么是xss

xss全程是跨站脚本，是一种注入式的攻击

xss成因

  • 对于用户输入没有严格控制而直接输出到页面
  • 对非预期输入的信任

xss分类

  • 存储型（持久型）
  • 反射型（非持久型）
  • DOM型

xss的危害

  • 盗取各类用户账号，如机器登录账号、用户网银账号、各类管理员账号
  • 窃取数据
  • 非法转账
  • 挂马
